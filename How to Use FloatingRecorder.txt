HOW TO USE FLOATINGRECORDER
===========================

🚀 QUICK START:
1. Double-click "FloatingRecorder.app"
2. If security warning appears: Right-click → Open → Confirm
3. Grant microphone permission when asked

🎙️ RECORDING:
• Click the record button in the floating window
• Or use global hotkey (Command+Control+R)
• Speak clearly into your microphone
• Click stop when finished

📝 TRANSCRIPTION:
• Text appears automatically after recording
• Transcription is copied to clipboard
• Paste anywhere with Command+V

📁 FILES:
• Audio recordings saved to: Documents/FloatingRecorder Transcriptions/
• App can run from anywhere (Desktop, Downloads, etc.)
• No installation to Applications folder required

⚙️ SETTINGS:
• Click preferences to change hotkeys
• Configure auto-paste behavior
• View transcription history

🔒 PRIVACY:
• All processing happens on your Mac
• No internet connection required
• Your voice data stays private

❓ TROUBLESHOOTING:
• App won't open? Right-click → Open (first time only)
• No transcription? Check microphone permissions
• Poor quality? Speak closer to microphone

Need help? Check the main README.md file for more details.
